The goal of r2dii.data is to host and document r2dii datasets, including
generic datasets, and fake datasets for examples and tests.

## Installation

Install the development version of r2dii.data with something like this:

``` r
# install.packages("devtools")
devtools::install_github("2DegreesInvesting/r2dii.dataraw")
```

errors?](https://gist.github.com/maurolepore/a0187be9d40aee95a43f20a85f4caed6#installation)
